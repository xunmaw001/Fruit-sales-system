# Fruit-sales-system
这款水果销售微信小程序，让水果采购便捷又新鲜。无需下载，在微信中即可开启选购。首页展示丰富水果品类，高清图片搭配产地、甜度、口感介绍。产地直发，保证新鲜。设置多种选购规格，整箱、斤两任你选。智能推荐热门水果，还有限时折扣、新人专享优惠。支持在线支付，物流信息实时追踪。果农能通过后台上架水果、更新库存、管理订单。无论是消费者尝鲜，还是果农拓宽销路，都十分便利 。
